07

Test plan

Test order: angle → hook → creative → audience. Don't optimize creative for a losing angle.

First test: 3 angles (Speed · Comparison · Outcome) × 2 hooks each = 6 ad sets, equal budget, one audience.

Decision metric: cost per trial start (not CTR). Rule: kill ad sets below 50% of the best after ~50 clicks; scale the winning angle, then test new hooks within it.

Next Refresh the competitor ad read (vantage-ads) monthly — when the set finally moves onto speed-to-value, change the angle before it gets crowded.
